\# Media Server Manager (Modernized Edition)



A fully modernized, Python‑based control panel for managing your home media server.  

Includes:



\- SSH connection panel (password or key)

\- Quick Commands panel

\- Systemd Services control

\- Docker container control

\- Real‑time system dashboard (CPU, RAM, Disk, Network)

\- Color‑coded console output

\- Animated collapsible sidebar

\- Persistent settings (host, username, sidebar state)



\---



\## 🚀 Features



\### ✔ SSH Connection

\- Auto‑detect SSH key  

\- Password optional  

\- Threaded connect/disconnect  

\- Shutdown button  

\- Persistent host + username  



\### ✔ Quick Commands

\- Grouped, scrollable, color‑coded  

\- Threaded execution  

\- Console logging  



\### ✔ Services Tab

\- Modern service cards  

\- Start / Stop / Restart / Logs  

\- Color‑coded status dots  

\- Auto‑refresh  



\### ✔ Docker Tab

\- Container cards  

\- Start / Stop / Restart / Logs / Inspect  

\- Auto‑refresh  



\### ✔ Console

\- Timestamped output  

\- Color‑coded  

\- Smart auto‑scroll  

\- Thread‑safe  



\### ✔ Dashboard

\- CPU usage  

\- RAM usage  

\- Disk usage  

\- Network throughput  

\- Auto‑refresh
